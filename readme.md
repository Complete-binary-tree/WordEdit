## 如何使用

- 如果是源码
  
  程序的入口文件是 `WordEdit.py`。请使用 `3.8.10`（或更高）的 `python` 运行。

- 如果是打包好的 `.exe` 文件

  该文件在 Windows 11 23H2 环境下打包（使用 Pyinstaller），可能在 Windows 7- 操作系统下无法使用。

  ~~什么？你说 Unix 系统？自己编译去吧，作者的电脑没有 Unix 系统！~~

然后你应该能看到一个 GUI 窗口。

- “开始背诵”：顾名思义，就是开始背诵。

  点进去后会随机一个你加入的单词，随机给英文/中文，让你写出另一个。

  ~~什么？你只想给中文写英语？别骂了别骂了，下个版本会有的~~

  写就是打字在文本框。

  写完以后点击“确定”按钮进行判定，结果显示在下方，同时重新随机单词，继续背诵。

  英文全文匹配，中文只要一个能对上就行（也就是说如果它的意思是“欢迎、你好”，你回答“好的、欢迎”是正确的。

- “添加单词”：可以添加单词。

  中文可以添加多个，不同的中文用空格隔开。

  下面的列表用于显示加的词汇和是否成功。

  点击按钮添加单词。

- “编辑单词”：表格显示的是已经添加的单词。

  选中表格的一（些）项，可以进行删除操作。

  选中表格的一项，可以进行修改操作。

  删除多项的时候可能会有些问题，请勿在未删除完之前进行其他操作！（后面会改进，如删除时禁止其他操作）

  文本框用来查询，中文/英文都可以，暂不支持查询中文的不同意思。

- “设置”：还没做。

- “退出”：和右上角关闭一个效果，保存所有的单词并退出。
